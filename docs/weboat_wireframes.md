# 📐 CONVENÇÕES DOS WIREFRAMES

```
┌─────────────┐  = Container/Seção
│             │
└─────────────┘

[BOTÃO]         = Botão clicável
[___________]   = Campo de input
(•) opção       = Radio button
☐ opção         = Checkbox
▼               = Dropdown/Select
🖼️              = Imagem/Foto
▶               = Vídeo
←  →            = Navegação/Carrossel
```
